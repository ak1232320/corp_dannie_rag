## Краткое описание

RAG поверх **критериев приемлемости клинических исследований** (датасет Kaggle
*Clinical Trial Eligibility Criteria*, источник — ClinicalTrials.gov, CC BY 4.0).
Повторён пайплайн rag-tutorial на своих данных и расширен.

- **Данные:** стратифицированная выборка **1200 испытаний** (по 150 на каждое из
  8 заболеваний) → **8585 чанков**. См. `doc/DATA.md`.
- **Пайплайн:** `datasets.json → documents.jsonl → chunks.jsonl → индекс
  (TF-IDF + эмбеддинги) → поиск → ответ с источниками`, UI на Streamlit.
- **Улучшения (3):** гибридный поиск (TF-IDF + мультиязычные эмбеддинги), оценка
  качества (retrieval@k / hit-rate / MRR, TF-IDF vs гибрид), LLM-генерация через
  OpenRouter с безопасным экстрактивным фолбэком. См. `homework/IMPROVEMENTS.md`.
- **Тесты:** 34 (`uv run pytest`).
- **Результат оценки:** hit@3 — TF-IDF 0.875 vs Hybrid **1.000**; на кросс-язычных
  запросах TF-IDF 0.125 vs Hybrid **1.000**; офтоп-вопрос корректно отклоняется
  только гибридом.
- **3 демо-вопроса + 1 негативный** (отказ) — в `README.md`.

> Окружение полностью локальное (`uv` + `.venv`), глобальные установки не требуются.
